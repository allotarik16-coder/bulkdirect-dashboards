# BulkDirect Dashboards

Admin, Client, and Public dashboards for BulkDirect agency platform.

## Versions

- **A: Dark Theme** - Modern, energetic (blue→cyan gradient)
- **B: Minimal** - Clean, professional (white background)
- **C: Corporate** - Enterprise, complete (blue professional)

## Setup

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

## Deploy to Netlify

Connected to GitHub Actions. Push to main branch = auto-deploy.
